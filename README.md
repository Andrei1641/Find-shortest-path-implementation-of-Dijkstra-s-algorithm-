# Control Instructions

## Create Node

To create a Node hover your mouse over the location where the Node will be created, and press P.
It can't be created near to another one.

## Connect Nodes

To connect two nodes select needed Nodes and press enter.

## Select Start/Finish

To select start position select only one point and press enter. To select end position select only one point and press enter(start position need to be selected).

## Search the shortest Path

To search the shortest path start and end point need to be selected and in the same net. After all
preparations press S.
